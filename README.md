# AI/ML Club Website (Vite + React)

A modern, responsive one-page university club website with a dark futuristic visual style.

## Tech Stack
- Vite + React
- CSS Modules

## Sections
- Sticky blurred navbar
- Hero with CTAs
- About with focus areas
- Upcoming events with glowing cards
- Join section
- Footer with social links

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```
